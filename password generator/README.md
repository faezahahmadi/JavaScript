# Password Generator

A simple password generator built with HTML, CSS, and JavaScript that allows users to generate random passwords based on their selected criteria.

# Features

1.	Generate random passwords instantly
2.	Choose password length
3.	Include lowercase letters
4.	Include uppercase letters
5.	Include numbers
6.	Include special symbols
7.	Input validation for password length
8.	User-friendly interface
9.	Responsive card-based design

# Technologies Used
1.	HTML5
2.	CSS3
3.	JavaScript (ES6)

# How It Works

1.	Enter a password length between 6 and 20 characters.
2.	Select one or more character types: 
o	Lowercase letters
o	Uppercase letters
o	Numbers
o	Symbols
3.	Click the Generate button.
4.	The generated password will be displayed on the page.

# Validation Rules

1.	Password length must be between 6 and 20 characters.
2.	At least one character type must be selected.
3.	Empty input is handled with an appropriate message.
