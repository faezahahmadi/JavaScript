#Password Generator
A simple password generator built with HTML, CSS, and JavaScript that allows users to generate random passwords based on their selected criteria.
#Features
•	Generate random passwords instantly
•	Choose password length
•	Include lowercase letters
•	Include uppercase letters
•	Include numbers
•	Include special symbols
•	Input validation for password length
•	User-friendly interface
•	Responsive card-based design
#Technologies Used
•	HTML5
•	CSS3
•	JavaScript (ES6)
#How It Works
1.	Enter a password length between 6 and 20 characters.
2.	Select one or more character types: 
o	Lowercase letters
o	Uppercase letters
o	Numbers
o	Symbols
3.	Click the Generate button.
4.	The generated password will be displayed on the page.
#Validation Rules
•	Password length must be between 6 and 20 characters.
•	At least one character type must be selected.
•	Empty input is handled with an appropriate message.
